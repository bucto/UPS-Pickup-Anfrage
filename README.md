# UPS Pickup Anfrage

Dieses Projekt ermöglicht die **Automatisierung von UPS Pickup-Anfragen** über die UPS API.  
Es wurde entwickelt, um wiederkehrende Abholaufträge effizient zu verwalten und den manuellen Aufwand zu minimieren.  

---

## Features

- Dynamische **Erstellung von Pickup-Anfragen** in JSON-Format.  
- Unterstützung für **mehrere Trackingnummern** pro Pickup.  
- **UTF-8 kompatibel** – funktioniert mit Umlauten und Sonderzeichen.  
- Automatische **Fehleranalyse und Status-Update** der Anfrage in der Datenbank.  
- Logging und **MySQL Integration** für Speicherung von Pickup-Daten und Antworten.  
- Anpassbar für **verschiedene Länder** und Service-Kategorien.

# UPS Pickup Request – Automation

This project automates **UPS pickup requests** via the UPS API.  
It is designed to efficiently manage recurring pickup tasks and reduce manual effort.

---

## Features

- Dynamic **creation of pickup requests** in JSON format.  
- Supports **multiple tracking numbers** per pickup.  
- Fully **UTF-8 compatible** – works with special characters and umlauts.  
- Automatic **error handling and database status updates**.  
- Logging and **MySQL integration** for storing pickup data and responses.  
- Configurable for **different countries and service categories**.
